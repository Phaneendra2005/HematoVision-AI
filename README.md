
# HematoVision: Advanced Blood Cell Classification Using Transfer Learning

## Overview
HematoVision is an AI-powered system for classifying blood cells using Transfer Learning and Deep Learning.
It classifies:
- Eosinophils
- Lymphocytes
- Monocytes
- Neutrophils

Dataset Size: 12,000 annotated images

## Features
- Transfer Learning (ResNet50)
- High Accuracy CNN Model
- Scalable Deployment Ready
- Medical + Educational Use Cases

## Installation
```bash
pip install -r requirements.txt
```

## Run Training
```bash
python train.py
```

## Run Prediction
```bash
python predict.py --image sample.jpg
```

## Applications
- Automated diagnostics
- Telemedicine
- Medical training platforms
